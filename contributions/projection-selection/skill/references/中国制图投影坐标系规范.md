# 中国制图投影坐标系规范

## 地理坐标系基准：CGCS2000

中国现行测绘成果统一采用 **2000 国家大地坐标系（CGCS2000，China Geodetic Coordinate System 2000）** 作为国家法定地理坐标系。全国所有新生产的测绘成果均要求使用 CGCS2000。

CGCS2000 是地心坐标系，其参考椭球参数与 WGS84 极为接近（椭球扁率仅在第 11 位有效数字上存在差异），在精度要求不高的场景下可近似等同于 WGS84。

> **历史沿革：** 中国此前曾使用北京 54（Beijing 1954）和西安 80（Xian 1980）两种参心坐标系。在处理历史数据时需注意坐标系的区分与转换。

本文档以下所有投影规范均基于 CGCS2000。

---

## 一、国家级制图

国家级制图指覆盖中国全境的地图。根据地图用途和版式，有以下三种标准投影方案。

> **重要：** 无论采用哪种方案，中国地图都**必须绘制南海九段线**，这是中国地图制图的基本规范要求。

### 1. 横版中国地图 — 阿尔伯斯等面积圆锥投影（推荐）

这是绘制**标准横版中国地图**的首选投影。

| 参数         | 值                                            |
| ------------ | --------------------------------------------- |
| 投影类型     | Albers Equal-Area Conic（阿尔伯斯等面积圆锥） |
| 中央经线     | 110°E                                         |
| 第一标准纬线 | 25°N                                          |
| 第二标准纬线 | 47°N                                          |

中央经线 110°E 大致位于中国领土的东西中线，两条标准纬线 25°N 和 47°N 分别穿过中国南部和北部，使得中国大部分区域的面积变形极小。

**特点：**

- 等面积，适合专题地图（人口密度、GDP 分布、土地利用等）
- 中国全境形状自然美观，与标准地图册中的呈现效果一致
- 两条标准纬线之间的区域变形最小

**适用场景：** 中国全域专题地图、统计数据可视化、学术论文配图。

> **注意：** 横版中国地图由于版面比例限制，南海诸岛无法在主图中完整呈现，必须在地图右下角添加**南海附图（南海小地图）**，以确保中国领土的完整性。

---

### 2. 气象/大气科学常用 — 兰伯特等角圆锥投影

中国气象领域最常用的投影方案，许多数值天气预报模式（如 WRF）默认采用此投影。

| 参数         | 值                                        |
| ------------ | ----------------------------------------- |
| 投影类型     | Lambert Conformal Conic（兰伯特等角圆锥） |
| 中央经线     | 110°E                                     |
| 第一标准纬线 | 25°N                                      |
| 第二标准纬线 | 47°N                                      |

参数设置与横版等面积方案一致，但投影性质不同。

**特点：**

- 等角，局部形状保持不变，风向、气压梯度等矢量场方向正确
- WRF、GRAPES 等气象模式的原生投影，数据可直接对齐绘图
- 不保面积，但中纬度区域面积畸变很小

**适用场景：** 天气预报图、气象数据可视化、WRF 模式输出绘图、风场/气压场分析。

---

### 3. 竖版中国地图 — 等距方位投影

用于绘制**竖版中国地图**。2014 年起中国官方开始推行竖版中国地图。

| 参数     | 值                                |
| -------- | --------------------------------- |
| 投影类型 | Azimuthal Equidistant（等距方位） |
| 中心经度 | 105°E                             |
| 中心纬度 | 35°N                              |

中心点 (105°E, 35°N) 大致位于中国领土的几何中心（甘肃兰州附近）。竖版地图将南海诸岛纳入主图，不再需要单独的南海附图框，完整呈现中国全部领土。

**特点：**

- 从中心点出发的距离和方向精确
- 竖幅布局，南海诸岛与大陆连为一体

**适用场景：** 竖版中国全图、国防与领土展示、需要完整呈现南海区域的场景。

---

### 三种方案对比

| 方案             | 投影类型                | 核心属性 | 最佳用途               |
| ---------------- | ----------------------- | -------- | ---------------------- |
| 横版中国（推荐） | Albers Equal-Area Conic | 等面积   | 专题地图、统计数据     |
| 气象常用         | Lambert Conformal Conic | 等角     | 天气图、气象模式       |
| 竖版中国         | Azimuthal Equidistant   | 等距     | 竖版全图、南海完整展示 |

## 二、省级制图

### 投影选择原则

绝大多数省级行政区呈中纬度、东西向延伸的形态，往往跨越多个高斯-克吕格 3° 投影带。高斯-克吕格投影跨带后边缘长度变形急剧增大，图幅拼接困难；而**双标准纬线圆锥投影**（含兰伯特等角圆锥、阿尔伯斯等面积圆锥），能对东西跨度大的中纬度区域实现全域变形均匀、无跨带割裂，整幅图的空间连续性和图形保真度远优于分带高斯投影。

> **例外情况：** 对于东西跨度极小、仅覆盖 1–2 个高斯投影带的省级行政区（如北京、天津、上海、海南等），省级全域制图也可采用高斯-克吕格投影，变形可控制在规范允许范围内，无需强制使用圆锥投影。

---

### 参数选取方法

兰伯特等角圆锥投影与阿尔伯斯等面积圆锥投影的参数取法完全一致，核心遵循**「全域变形最小、分布最均匀」**的总原则，严格执行国家测绘行业通用的**「1/6 法则」与「几何中心法则」**。

**中央经线（几何中心法则）：**

$$L_0 = \frac{L_{min} + L_{max}}{2}$$

其中 $L_{min}$ 和 $L_{max}$ 分别为全域最西端和最东端经度。

> **取整规则：** 计算结果**必须**就近取整至 0.5° 或 1°，方便参数通用。取整偏差不超过 30′ 时，对变形的影响可忽略不计。

**标准纬线（1/6 法则）：**

1. 计算全域纬差：$\Delta\varphi = \varphi_{max} - \varphi_{min}$（$\varphi_{max}$ 为最北端纬度，$\varphi_{min}$ 为最南端纬度，北纬为正）
2. 南标准纬线（第一标准纬线）：$\varphi_1 = \varphi_{min} + \Delta\varphi \,/\, 6$
3. 北标准纬线（第二标准纬线）：$\varphi_2 = \varphi_{max} - \Delta\varphi \,/\, 6$

> **取整规则：** 计算结果**必须**就近取整至 0.5° 或 1°，方便参数通用。取整偏差不超过 30′ 时，对变形的影响可忽略不计。

## 三、省级以下制图（地市级、县级及更小区域）

中国绝大多数地市东西跨度 1°–6°（覆盖 1–2 个高斯-克吕格 3° 带），南北跨度一般不超过 5°。省级以下制图分为两大场景，投影选择差异显著。

### 场景一：全域中小比例尺通用制图

地市全域政区图、交通图、国土空间总体规划总图、资源环境专题图等，通常跨 2 个及以上高斯 3° 带，无高精度工程量测需求，核心需求是**全域图形保真、无跨带割裂**。

**推荐投影：** 双标准纬线兰伯特等角圆锥投影（或阿尔伯斯等面积圆锥投影），参数取法与省级制图完全一致（1/6 法则 + 几何中心法则）。

### 场景二：大比例尺法定测绘

城市规划详图、地籍图、工程测量图等大比例尺法定制图，对精度要求极高。

**推荐投影：** 高斯-克吕格 3° 带投影，CGCS2000 基准，严格执行国家 3° 带分带规则，直接套用对应带号的标准中央经线（$L_0 = 3° \times n$，$n$ 为 3° 带带号）。

> **特殊调整规则：** 若测区跨带、或因高程导致长度变形超过 2.5 cm/km 红线，可采用**自定义中央经线的任意带高斯投影**，中央经线取测区几何中心经线，以将变形控制在允许范围内。

### 按比例尺选择投影的决策规则

```
比例尺 ≥ 1:1万（大比例尺）
  └── 高斯-克吕格 3° 带投影，CGCS2000 基准，套用国家标准分带参数

比例尺 1:1万 ~ 1:10万（中比例尺）
  ├── 区域跨 ≥ 2 个 3° 带 → 兰伯特等角圆锥投影
  └── 区域在 1 个 3° 带内 → 高斯-克吕格 3° 带投影

比例尺 ≤ 1:10万（小比例尺）
  ├── 区域跨度大 → 兰伯特等角圆锥投影
  └── 区域跨度小 → 高斯-克吕格 3° 带投影
```

> **简单总结：** 大比例尺精确测绘一律用高斯-克吕格；中小比例尺看是否跨带——跨带用圆锥投影，不跨带用高斯-克吕格即可。

### 案例

**案例 1：武汉市全域政区图（地市级，1:25 万）**

- 全域边界：最西端 113°41′E，最东端 115°05′E；最北端 31°22′N，最南端 29°58′N
- 跨 2 个高斯 3° 带，属于中小比例尺全域制图场景
- **选型：** 兰伯特等角圆锥投影
- **参数：** 中央经线取全域几何中心 114.5°E；纬差 1°24′，标准纬线取整为 30°N、31°N

**案例 2：武汉市武昌区规划图（县级，1:1 万）**

- 全域边界：完全落在高斯 3° 带第 38 带（中央经线 114°E）内，东西跨度＜0.5°
- 大比例尺法定制图场景，无需圆锥投影
- **选型：** 高斯-克吕格 3° 带第 38 带投影，CGCS2000 基准
- **参数：** 直接套用国家分带参数，中央经线 114°E，东偏移 500000 m

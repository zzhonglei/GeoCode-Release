# China Statistics Data

Guidance for finding and downloading official thematic statistics from the National Bureau of Statistics of China.

## What it does

This skill helps GeoAgent collect tabular thematic data from official Chinese statistical sources, especially the National Bureau of Statistics data portal and China Statistical Yearbook.

Because the NBS data portal is browser-based, GeoAgent must have Playwright MCP enabled before it can operate the website and download data. If Playwright MCP is not available, the skill instructs GeoAgent to ask the user to enable it from GeoCode's top-right status button.

Typical tasks include downloading or compiling indicators such as GDP, population, employment, industry, agriculture, energy, environment, transport, education, and other socio-economic statistics for maps, charts, reports, or GIS joins.

## Main sources

- National Bureau of Statistics data portal: <https://data.stats.gov.cn/>
- National Bureau of Statistics data index: <https://www.stats.gov.cn/sj/>
- China Statistical Yearbook: <https://www.stats.gov.cn/sj/ndsj/>

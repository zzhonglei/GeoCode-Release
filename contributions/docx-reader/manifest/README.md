# DOCX Skill

Create, read, edit, convert, and review Word/`.docx` documents.

This skill is based on the K-Dense scientific-agent-skills `docx` skill. It
supports professional Word document workflows, including:

- Creating new `.docx` files with headings, tables, images, headers, footers,
  page numbers, footnotes, hyperlinks, and table of contents
- Reading and analyzing existing Word documents
- Editing existing `.docx` files by unpacking, modifying Office XML, validating,
  and repacking
- Working with tracked changes and comments
- Converting legacy `.doc` files to `.docx`
- Converting Word documents to PDF/images for visual inspection

The bundled scripts include Office unpack/pack/validate helpers, tracked-change
acceptance, comment insertion, and schema validation support.

Source: <https://github.com/k-dense-ai/scientific-agent-skills>
